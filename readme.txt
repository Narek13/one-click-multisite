Easily convert your single-site WordPress into a multisite network with one click.

== Description ==
This plugin automatically configures your wp-config.php and .htaccess to enable WordPress multisite, saving you from manual editing.

== Installation ==
1. Upload the plugin and activate it.
2. Go to Tools → Network Setup.
3. Click the "Finish Multisite Setup Automatically" button.
4. You will be redirected to the new multisite dashboard.
